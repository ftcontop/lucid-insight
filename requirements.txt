# In your local project folder:
git add requirements.txt
git commit -m "Added groq package for AI chat"
git push origin main
```

Then Railway will automatically:
1. Detect the change
2. Install `groq>=0.4.0`
3. Redeploy with AI chat working

## Or if you're lazy:

Just manually edit `requirements.txt` on GitHub:

1. Go to your repo on GitHub
2. Click `requirements.txt`
3. Click the pencil icon (edit)
4. Change it to:
```
discord.py>=2.0.0
aiohttp>=3.8.0
python-dotenv>=1.0.0
groq>=0.4.0
